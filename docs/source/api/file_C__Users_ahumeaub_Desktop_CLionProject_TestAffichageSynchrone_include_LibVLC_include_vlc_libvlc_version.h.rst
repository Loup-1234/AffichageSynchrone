Detailed Description
--------------------

This file defines version macros for LibVLC. Those macros are primilarly intended for conditional (pre)compilation. To get the run-time LibVLC version, use :ref:`exhale_function_group__libvlc__core_1ga130ea683276a8504d445d18d804720e5` instead (the run-time version may be more recent than build-time one, thanks to backward binary compatibility).

This header file is available in LibVLC 1.1.4 and higher.
